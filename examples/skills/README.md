# Examples Skills

В этой папке лежат примеры skills для Codex:

- `python-bugfix/` — узкий workflow для безопасного исправления Python-багов.
- `repo-docs/` — skill для генерации документации по существующему репозиторию.

Оба примера показывают минимальную структуру:

- директория skill;
- файл `SKILL.md`;
- четкое назначение;
- пошаговый workflow;
- ограничения и validation.

Используй эти примеры как шаблоны, а не как готовые universal skills.
